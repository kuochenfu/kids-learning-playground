# 🛠️ Development Log: 2026-03-01 💎

## 🚀 Overview
Today's mission was to transform the "Science Quiz" and "Logic Puzzles" into high-quality educational experiences. We focused on eliminating redundancy, adding modern Computer Science (CS) content, and fixing critical navigation bugs.

---

## ✨ Features & Enhancements

### 1. 🧬 High-Quality Question Bank (Version 10)
*   **Source Transformation**: Replaced bulk-generated fillers (e.g., "Is 208 a prime number?") with **100 hand-curated, high-quality questions**.
*   **CS Integration**: Added 50 Science questions focusing on AI, Drones, Robotics, Hardware (CPU/SSD), and basic Coding (Binary).
*   **Logic Enrichment**: Added 50 classic riddles and logic sequences (e.g., "What has a face but no legs?") to train logical thinking for Grades 4-6.
*   **Educational Facts**: Every question now includes a "Did you know?" fact based on real-world knowledge.

### 2. 🎲 "Zero-Repetition" Engine
*   **Triple Shuffling**:
    1.  **Database Level**: GORM `Order("RANDOM()")` ensures a random set is fetched from the server.
    2.  **Frontend Level**: Implemented `shuffleArray` (Fisher-Yates) to shuffle the already random set on the client.
    3.  **Cache-Busting**: Appended `_t=${Date.now()}` to all API calls to prevent browser caching of old question sets.

## 🐛 Bug Fixes & Technical Debt

### 1. 🛑 "Return to Lobby" Critical Fix
*   **The Issue**: The "Return" button in the empty bank or error states only updated `gameState: 'idle'`, which kept the user trapped on the same page.
*   **The Fix**: Integrated `useNavigate` from `react-router-dom`. All return triggers now force a navigation back to the `/lobby` map.
*   **New UI**: Added a secondary "Return to Lobby" button on the **Game Finished** screen for better UX flow.

### 2. 🔌 Database Force-Seeding (V10)
*   **The Issue**: Render deployments sometimes kept old DB records, preventing new questions from appearing.
*   **The Fix**: Implemented **Strict Version Check**. The backend now searches for a specific `meta` record (Version V10). If it's missing or the count is wrong, it drops the entire `questions` table and performs a fresh seed.

## 🐛 Bug Fixes & Technical Debt (Session 2)

### 1. 🔄 List Prepending Fix
*   **The Issue**: Freshly uploaded images were displacing the default kitten image in the list.
*   **The Fix**: Modified the `/api/puzzles` endpoint to explicitly **prepend** canonical assets before scanning the `uploads` directory.

### 2. ⚡ Scroll Alignment Fix
*   **The Issue**: Clicking the left/right arrows or selecting a card felt "jumpy" and didn't align properly.
*   **The Fix**: Overhauled the carousel layout with `snap-center` CSS and React `useRef`. The container now calculates the offset to keep cards perfectly balanced.

### 3. 📁 Upload Persistence
*   **Recommendation**: Documented the need for **Persistent Volumes (PV)** for the `./backend/uploads` directory in cloud environments to prevent data loss during container restarts.
